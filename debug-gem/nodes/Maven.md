# Maven

Build tool and dependency manager for [[Java]]. Uses `pom.xml`. Central repository at search.maven.org.

## Common Gotchas
- Local `.m2` cache corruption — delete and rebuild
- SNAPSHOT versions can change without version bump — pin in production
- Use `./mvnw` (Maven Wrapper) for consistent versions
