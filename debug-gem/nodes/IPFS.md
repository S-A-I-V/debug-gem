# IPFS (InterPlanetary File System)

Decentralized storage network. Content-addressed (CID-based). Used for NFT metadata, dApp assets.

## Known For These Error Patterns
- [[Connection & Network Errors]] — CID not found, gateway timeout
- [[Resource Exhaustion Errors]] — Large DAG traversal timeout

## Key Concept
Content must be "pinned" to stay available. Unpinned content may be garbage collected.
