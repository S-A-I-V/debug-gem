# Gradle

Build tool for [[Java]]/[[Kotlin]]. Groovy or Kotlin DSL. More flexible than [[Maven]], used by Android.

## Common Gotchas
- Gradle daemon persists between builds — can accumulate memory (`gradle --stop`)
- Use `./gradlew` (Gradle Wrapper) for consistent versions
